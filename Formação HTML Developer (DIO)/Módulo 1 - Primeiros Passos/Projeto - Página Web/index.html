1. O que é a Internet e como ela funciona?

De maneira simples, podemos definir a internet como uma rede global de computadores que permite a troca de informações entre usuários em todo o mundo. Ela utiliza protocolos padronizados e diversos meios de transmissão, como cabos de fibra óptica e redes sem fio, para conectar dispositivos eletrônicos. A Internet oferece uma ampla gama de serviços, como e-mail, navegação na web, redes sociais e streaming de mídia, desempenhando um papel fundamental na comunicação, educação, entretenimento e negócios.

2. O que são Clients e Servers?

Um Cliente ou Client é um dispositivo ou programa que solicita serviços ou recursos de outro dispositivo chamado servidor. Em um contexto de Internet, o cliente é geralmente o software em um computador, tablet, smartphone ou outro dispositivo que acessa serviços online, como navegadores da web, clientes de e-mail ou aplicativos de mensagens.

Já um Server ou Servidor é um dispositivo ou software que fornece serviços, recursos ou dados a outros dispositivos chamados clientes. Servidores são projetados para atender a solicitações dos clientes, fornecendo informações, processamento de dados, armazenamento de arquivos ou execução de outras tarefas. Na Internet, servidores podem ser usados para hospedar sites, e-mails, jogos online, armazenar dados em nuvem e realizar diversas funções.

3. O que são as Aplicações Web?

As Aplicações Web, resumidamente, são soluções digitais criadas que possuem a internet como meio de comunicação entre Client x Server. Assim, uma Aplicação Web não requer instalação em uma máquina, já que atuam diretamente na internet. Desse modo, podemos conluir, que os Programas e Aplicativos instalados nos dispositivos não são considerados uma Aplicação Web, mesmo que dependam da internet para cumprirem suas funcionalidades.

Exemplo: LinkedIn, esta rede social focada em temática profissionais possui a sua versão Aplicação Web, bem como a sua versão Aplicativo.

4. O que é o HTML?

HTML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), é uma linguagem de marcação usada na construção e estruturação de páginas web. Ele é fundamental para criar documentos web, permitindo a criação de páginas que podem ser visualizadas e integradas em navegadores da web.

Curiosidade: O HTML não é considerada uma linguagem de programação, apenas de marcação.
